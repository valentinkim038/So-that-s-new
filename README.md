## Списки
Чтобы выделить не нумерованный список, используйте (*) или знаком (+):
* элемент 1
* элемент 2
* элемент 3
+ Элемент 4
Чтобы добавить нумерованные списки, необходимо просто пронумеровать:
1. Элемент 1
2. Элемент 2
3. Элемент 3
